


                              !!! В этом проекте он подключен к REST контроллеру "(Theme) Spring, Web Development/ToDoList with Redis" !!!



Установка:

1) скачать с сайта архив, разархивировать его
2) закинуть папку cd redis-6.0.10 в терминал
3) make



Запустить:

1) закинуть папку cd redis-6.0.10 в терминал
2) src/redis-server     - это запустит сервер
3) Открыть ещё одно окно в терминале, также закинуть туда папку
4) src/redis-cli        - это подключит к серверу, и можно использовать уже команды









Команды:

SET name egor     - создать, ключ name значение egor 
GET name          - получить значение по ключу
DEL name          - удалить по ключу (любой ключ так удаляется)


Set:

SADD courses Java Python Data    - добавить по ключу courses 3 уникальных значения, елси добавлять ещё, будут добавляться, а не перезаписываться
SMEMBERS courses                 - получить по ключу
SREM courses Data                - удалить значение в сете по ключу


HashMap:
HSET Egor name "Egor" age "26"   - по уникальному ключу Egor добавить 2 поля, name: egor и age: 26
HGET Egor name                   - получить значение поля name из хэшмапы Egor (получим: "Egor")
HMGET Egor name age              - то же самое, только для нескольких полей (получим: "Egor" и "26")
HKEYS Egor                       - получить все названия полей из хэшмапы с ключом Egor (получим: "name" и "age")
HDEL Egor age                    - удаляет из мапы Egor одно из полей вместе со значением



Array:
LPUSH colors red               - добавить "red" в лист colors (или создать если не сущ) в конец
LRANGE colors 0 -1             - показать все элементы списка (LRANGE colors 0 0 - покажет только 0 индекс)
LREM colors 0 red              - удалить из colors элемент red (0 означает удалить все элементы со значением red)
